== Description ==
Fervent Readability Optimizer provides advanced search readiness and content health scores.

== External Services ==
This plugin connects to an external API to analyze content for search engine readiness and readability diagnostics. 
- Service Endpoint: https://api.getrainos.com/v1
- Data Transmitted: Content titles, plain-text body copy, and target post permalinks are transmitted securely over HTTPS upon manual request or post optimization cycles.
- Terms & Privacy: This service is provided by Rain OS. You can review the respective Terms of Service (https://www.rainos.com/terms) and Privacy Policy (https://www.rainos.com/privacy) for data handling details.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/fervent-readability-optimizer` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
